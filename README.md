# Installation Process on Termux
<b>Copy entire this code and paste on termux👇 if ask for any Yes or No then just tap on Enter<b>
```shell
apt update && apt upgrade -y
apt install git
git clone https://github.com/Ttechnical/pdflocked.git
cd pdflocked
chmod 777 setup.sh
./setup.sh
```
# Usage
### Step1
